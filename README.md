#### Q: TinyMCE settings is displayed inside the EvoFLAT-SystemSettings tab

R: 
- Go to **Elements > Plugins > Edit Plugin Execution Order > OnInterfaceSettingsRender** 
- Drag **EvoFLAT-SystemSettings** Plugin BELOW **TinyMCE** 
- Save
